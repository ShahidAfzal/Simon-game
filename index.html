How to Play Simon

  1. Press any key to start the game
  2. Simon will play a sequence of colors
  3. Repeat the sequence by clicking the colors in the same order
  4. Each successful round adds one more color to the sequence
  5. If you make a mistake, the game ends
  6. Try to beat your highest score!

Mobile users: Tap anywhere on screen to start
